# Парсер сайта https://langwitch.ru/

При обучении на сайте [netology.ru](https://netology.ru/), есть задание по созданию ТГ-бота тренажера по английскому языку. 

По условию задания, нужно будет предлагать русское слово и несколько вариантов на английском. Хранение слов и всех данных должно осуществляться в PostgreSQL. Данные по словарю можно было наполнить вручную самому, но я задался целью попробовать найти такие данные в интернете и собрать словарь.

[Сайт](https://langwitch.ru/) отлично подошел для знакомства с библиотекой bs4 и первым изучением навыков парсинга.

Результат выполнения этой программы файл с категориями и наполнением этих категорий словарем где ключ - русское слово, а значение его перевод на английский.

### Запуск программы

Сначала установите зависимости 

>pip install -r requirements.txt

Запуск программы 

> main.py

Результат выполнения в папке ***'data/'***

> [vocabulary.json](data/vocabulary.json)
