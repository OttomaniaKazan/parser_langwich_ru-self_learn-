1. модуль fetcher.py описывает функцию запроса к странице. В качестве обработчика GET-запроса, выступает модифицированный класс, написанный мной же для другой программы. Сам класс находится в модуле request_handler.py
2. модуль возвращает объект типа str
3. в папке test есть файл test_fetcher.py в котором тестово вызывается написанная функция на конкретном url. Так же там в заметках описан используемый заголовок в этой функции.
4. проверка осуществлялась вызовом функции в тестовом файле через консоль
